# Projet Mascarade

## Project setup
```
cd client
npm install
cd ../server
npm install
```

### Start server for API
In server folder :
rename .env.example by .env
```
npm start
```
### Start client
In client folder :
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```
